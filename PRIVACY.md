# Privacy Policy for IsThisAI

**Last Updated:** December 8, 2024

## Overview
IsThisAI is committed to protecting your privacy. This browser extension is designed with privacy-first principles and operates entirely within your browser.

## Data Collection
**We do NOT collect, store, or transmit any user data.**

Specifically:
- ❌ No personal information is collected
- ❌ No browsing history is tracked
- ❌ No text or images you analyze are sent to external servers
- ❌ No analytics or tracking tools are used
- ❌ No cookies are stored
- ❌ No user accounts or authentication required
- ❌ No advertising or monetization

## How the Extension Works
All analysis happens **locally in your browser**:
- Text analysis uses client-side JavaScript algorithms
- Image analysis processes files in browser memory only
- No data ever leaves your device
- No internet connection required for core functionality

## Permissions Explained
The extension requests the following browser permissions:

### activeTab
- **Purpose:** To analyze selected text on the current webpage
- **Usage:** Only when you explicitly right-click and choose "Check with IsThisAI"
- **Data Access:** Temporary access to selected text only
- **Not Used For:** Tracking, monitoring, or collecting browsing data

### contextMenus
- **Purpose:** To add the "Check with IsThisAI" option in right-click menu
- **Usage:** Displays menu item when text is selected
- **Data Access:** None

### scripting
- **Purpose:** To inject the analysis result modal on webpages
- **Usage:** Only when you request analysis via context menu
- **Data Access:** None beyond displaying results

### host_permissions: <all_urls>
- **Purpose:** To work on any website you visit
- **Usage:** Enables context menu functionality across all sites
- **Data Access:** Only text you explicitly select and analyze
- **Not Used For:** Background monitoring or data collection
